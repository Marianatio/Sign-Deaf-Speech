About App

Sign Deaf Speech

Provide sign language and speech recognition features to facilitate easy and effective communication between deaf and hearing individuals, a system can where sign language used by deaf individuals can be transcribed into text for hearing individuals to understand. Additionally, spoken messages are instantly converted into text to allow easy communication in one place.

App Features

Core Features

Real-Time Communication

Facilitating communication between deaf and hearing people using deep learning in sign language and speech recognition.

Standalone Sign Language Creation

Enhance user creation in creating sign languages and converting various other types of sign languages to BISINDO format.

BISINDO Sign Language Training

Provides BISINDO sign language alphabet learning features through interactive videos and quizzes.

Benefit

User Friendly interface and Easy To Use SDS App

Medium of communication between deaf and hearing people in daily life.

Providing deaf people with interactive communication accessibility to normal individuals in a variety of everyday situations effectively.
